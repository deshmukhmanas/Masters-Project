# Masters-Project
This is an project on Facial Emotion Recognition using CNN and SVM Machine Learning algorithms which I have done while doing masters degree.
In this project I have taken facial emotions data and I am comparing it with already available data of facial emotions.
I have performed analysis on facial data by using CNN(Convolution Neural Network Algorithm) and compared it with SVM(Support Vector Machine ) algorithm by checking parameters such as  f1 score,accuracy,precision,
At the end I got conclusion that SVM algorithm performs well and gives more accuracy than CNN algorithm.
