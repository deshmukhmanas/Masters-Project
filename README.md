# Facial Emotion Recognition using Machine Learning

## 📌 Overview
This project focuses on **Facial Emotion Recognition (FER)** using Machine Learning techniques. It predicts human emotions from facial images and compares the performance of two algorithms:
- Convolutional Neural Network (CNN)
- Support Vector Machine (SVM)

The goal is to determine which model performs better in terms of accuracy and prediction capability.

---

## 🎯 Objectives
- Detect human emotions from facial expressions
- Implement and evaluate CNN and SVM models
- Compare performance of both algorithms
- Analyze predicted vs actual emotions

---

## 📊 Dataset
- **FER-2013 Dataset** (Kaggle)
- 48x48 grayscale facial images
- 7 emotion classes:
  - Angry
  - Disgust
  - Fear
  - Happy
  - Sad
  - Surprise
  - Neutral

---

## ⚙️ Technologies Used
- Python
- Jupyter Notebook
- NumPy, Pandas
- Matplotlib, Seaborn
- Scikit-learn (SVM)
- TensorFlow / Keras (CNN)

---

## 🧠 Models Implemented

### 1. Convolutional Neural Network (CNN)
- Deep learning model for image classification
- Automatic feature extraction
- Higher accuracy for complex patterns

### 2. Support Vector Machine (SVM)
- Supervised learning model
- Used with:
  - Linear Kernel
  - RBF Kernel

---

## 📈 Results
- CNN achieved strong performance in emotion classification
- SVM performed well on smaller feature sets
- Comparative analysis done using:
  - Accuracy
  - Loss graphs
  - Confusion matrix
